# Build Push Sleep Static Site Demo

This example demonstrates the Build, Push, Sleep (BPS) philosophy by generating a simple static HTML file. The Python script first 'builds' the HTML content, then simulates the 'push' phase by making the file ready for deployment, allowing the developer to 'sleep' knowing the artifact is prepared.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run `python main.py` in your terminal.
3. Open the generated `index.html` file in your web browser, or run `python -m http.server` in the directory and navigate to `http://localhost:8000`.
